Minecraft Plugin for 1.20+ with Challenges

## Installation
1. Download the latest version of the [Plugin](https://github.com/TgZ39/ChallengePlugin/releases/).
2. Put the Plugin the the `plugins` folder of your Server.
3. Start the Server.
